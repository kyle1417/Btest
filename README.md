# 2025 安全员B证 · 在线刷题

面向「建筑施工企业安全生产管理人员（安全员B证）」考试的纯静态刷题应用。无框架、无构建，开箱即用。

🔗 在线地址：<https://b.pantao.online/>

## 功能

- **练习模式**：答完即时对错 + 解析
- **考试模式**：限时模拟卷、交卷算分
- **历年真题**：2024 / 2025 整卷模拟
- **错题本 / 收藏夹**：自动归集、随时重做
- **题目搜索**：按关键词查题
- **云同步**：基于 Cloudflare Worker + KV，跨设备同步进度/错题/收藏

题库共 **2000 题**，含单选、多选、判断三种题型。

## 项目结构

```
├── design-demos/       # 早期设计稿

```

## 本地预览

纯静态站点，任意静态服务器即可：

```bash
python -m http.server 8137
# 浏览器打开 http://localhost:8137/
```

## 部署

部署到 **Cloudflare Pages**（项目名 `btest`）。将 `index.html`、`css/`、`js/` 三者作为静态根目录上传即可：

```bash
# 需自备 CLOUDFLARE_API_TOKEN 环境变量
npx wrangler pages deploy <静态目录> --project-name btest --branch main
```

云同步后端单独部署在 `worker/`（`npx wrangler deploy`，需绑定一个 KV 命名空间）。

## 题库修订记录

原始题库由 PDF 教材 OCR 提取，约 2000 题，存在一定比例的答案错误与「答案 ↔ 解析对不上」的问题（解析多为早期模板批量生成，与答案脱钩）。

以**一份外部参考题库**配合 **29 部法规语料库**为校订依据，分批核订：

| 批次 | 数量 | 处理 |
|---|---|---|
| 解析可自证的分歧 | 273 条 | 参考解析能反推出答案、与参考一致 → 修正答案 |
| 语料检索复核 | 6 条 | 法规语料命中直接依据 → 修正答案 |
| 剩余分歧题 | 169 条 | 语料无直接依据，整体采信参考答案（并对其中约 24 道做了联网/法规抽验，结果 100% 与参考一致）|
| 解析重写 | 1535 条 | 垃圾模板解析替换为带法规出处的实质解析，消除「答案 ↔ 解析错配」|
| 漏网分歧题 | 6 条 | 补修答案 + 解析 |

- 累计 **459 条答案变更**，逐题记录见 [`169条答案修订清单.md`](./169条答案修订清单.md)。
- 其中第 694、874 两题经人工复核后保留原题库答案。

> 说明：自动化校订仍可能残留个别误判，发现问题欢迎反馈。
